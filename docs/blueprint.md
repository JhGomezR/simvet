# **App Name**: SimVet Urgencias

## Core Features:

- Patient Information Display: Display patient details like species, age, weight, and chief complaint in the left panel.
- Dynamic Vital Signs Monitoring: Real-time display of vital signs (heart rate, respiratory rate, temperature, perfusion, consciousness) in the right panel.
- Clinical Decision-Making Interface: Interactive buttons for ABCDE assessment, options for questioning, exam requests, and treatment choices in the central panel.
- Triage Performance Indicator: Generative AI assesses student triage skill and provides feedback
- Academic Feedback and Scoring: Provide detailed feedback on critical errors (red), correct decisions (green), ideal clinical pathways, and overall scoring.
- Professor Analytics Dashboard: Provide an analytical dashboard in the 'Professor Mode', containing performance visualization per cohort, common error mapping, comparative student reports and semesterly longitudinal evolution reports.

## Style Guidelines:

- Primary color: Medical blue (#29ABE2) to evoke trust and a clinical setting.
- Background color: Light gray (#F0F0F0) for a clean and unobtrusive backdrop.
- Accent color: Light blue (#72DDF7) to highlight key interactive elements.
- Body and headline font: 'Inter' for a modern, objective, neutral feel.
- Use minimalist, clinically relevant icons for each action and vital sign.
- A structured three-panel layout (Patient Info, Decisions, Monitoring) to guide the user through a clear clinical workflow.
- Subtle transitions and visual cues when vitals change or actions are performed, alerting the user to important changes in the simulated environment.
